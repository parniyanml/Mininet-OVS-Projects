# Computer Networks Laboratory Reports

This repository contains comprehensive technical reports and analysis for the **Computer Networks** course at the University of Isfahan.

## Contents
The PDF files include practical experiments and analysis of:
* **Software Defined Networking (SDN):** Simulation of topologies using **Mininet**.
* **Traffic Analysis:** Deep packet inspection using **Wireshark** (DNS, HTTP, TCP).
* **Network Configuration:** Implementation of Linux Namespaces and Open vSwitch (OVS).
